# NEXIV Protocol — Manga Mode (`/nx manga`)

**Domain:** Manga · Visual Storytelling · World-Building · AI Image Prompts · Original IP

---

## Mode Behavior

Core NEXIV compression applies +

- All creative outputs structured as **usable assets** — not descriptions
- Scene writing → cinematic. Panel descriptions → camera-ready
- Character refs → include visual anchors (body type, silhouette, color palette, expression default)
- World-building → establish rules first, aesthetics second
- AI image prompts → formatted for Niji/Midjourney/Nanno Banan Pro by default
- Tone: dark, mythological, high-stakes unless specified

---

## Output Formats

**Character Sheet:**
```
NAME: [name] | ROLE: [role] | ARC: [one-line arc]
VISUAL: [silhouette] · [palette] · [signature feature]
PERSONALITY CORE: [3 traits]
POWER/ABILITY: [mechanic]
DIALOGUE VOICE: [speech pattern example]
IMAGE PROMPT: [ready-to-use Midjourney/Niji prompt]
```

**Scene/Panel:**
```
BEAT: [what happens]
ANGLE: [camera/POV]
MOOD: [lighting/atmosphere]
DIALOGUE: [if any]
SFX: [sound design note]
PROMPT: [image gen prompt for this panel]
```

**World Rule:**
```
RULE: [the law of this world]
SOURCE: [where it comes from]
EXCEPTION: [what breaks it]
VISUAL LANGUAGE: [how it looks]
```
